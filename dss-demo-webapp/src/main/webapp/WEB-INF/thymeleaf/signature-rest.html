Los siguientes servicios web son accesibles mediante APIs (Interfaz de Programación de Aplicaciones) todas siguiendo desde la versión 5.6 las Especificaciones de OpenAPI. A demás, ocultan la complejidad de la firma (CAdES, PAdES, XAdES, ASiC) proporcionando una integración más sencilla.

Funciones disponibles:

  • Firma
    • Calcular la función hash de la información a firmar (getDataToSign);
    • Incorporar la firma digital al documento final (signDocument);
    • Extender una firma existente;
    • Validar una firma;
    • Generar un sello de tiempo.
  • Firmado del servidor
    • Obtención de las claves de un token;
    • Obtención de las claves de un token;
    • Firmar la información con una clave usando un alias;
    • Firmar la función hash con una clave usando un alias.
  • Validación de documentos
    • Valida un documento (firmado o con sello de tiempo);
    • Recuperar un documento firmado original.
  • Validación de certificados
    • Validar un certificado.
  • Sello de tiempo remoto
    • Marcar temporalmente un valor hash.
