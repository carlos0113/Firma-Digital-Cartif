Esta es la aplicación web de firma digital de Cartif.

Funcionalidades

En la actualidad, la aplicación posee las siguientes funcionalidades:

  • Firma de documento(s) en diversos formatos (XAdES, CAdES, PAdES, JAdES and ASiC-S/ASiC-E);
  • Firma de función hash;
  • Firma de documentos PDF;
  • Extensión de firma;
  • Marcado temporal de documentos;
  • Validación de documentos (firma y marca de tiempo);
  • Validación de certificados;
  • Obtención de tokens (para validación de documentos);
  • Validación de datos de diagnóstico;
  • Resumen EU LOTL;
  • Certificados de confianza OJ;
  • Aplicación Java independiente;
  • Servicios web REST/SOAP.

Por favor, consulte el manual de usuario, accesible desde la sección de "Documentación" en el menu situado a la izquierda.
