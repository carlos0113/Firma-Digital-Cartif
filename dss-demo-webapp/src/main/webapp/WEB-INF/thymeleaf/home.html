Este servicio de Cartif proporciona la diversas funcionalidades relacionadas con la firma digital de documentos, haciendo uso de la libreria de codigo abierto Digital Signature Services (DSS) de OASIS, respaldado por la Comisión Europea.

Todas las funciones de esta aplicación web cumplimenta la legislación vigente europea eIDAS, un conjunto de normas para la identificación electrónica y los servicios de confianza para transacciones electrónicas en el mercado único europeo.

Funcionalidades

En la actualidad, la aplicación posee las siguientes funcionalidades:

  • Firma de documento(s) en diversos formatos (XAdES, CAdES, PAdES, JAdES and ASiC-S/ASiC-E);
  • Firma de función hash;
  • Firma de documentos PDF;
  • Extensión de firma;
  • Marcado temporal de documentos;
  • Validación de documentos (firma y marca de tiempo);
  • Validación de certificados;
  • Obtención de tokens (para validación de documentos);
  • Validación de datos de diagnóstico;
  • Resumen EU LOTL;
  • Certificados de confianza OJ;
  • Aplicación Java independiente;
  • Servicios web REST/SOAP.
